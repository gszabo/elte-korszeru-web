Ajánlott irodalom

Agilis szoftverfejlesztés

  • Kent Beck: Extreme Programming Explained - Embrace Change
    Könnyed, rövid könyvében az XP atyja összefoglalja a lényegét.
    https://www.amazon.com/Extreme-Programming-Explained-Embrace-Change/dp/0321278658
  • James Shore, Shane Warden - Art of Agile Development
    Részletes referencia az XP alkalmazásához.
    https://www.amazon.com/Art-Agile-Development-Pragmatic-Software/dp/0596527675
  • Uncle Bob (Robert C. Martin) visszatekintője az XP sikerére (blog poszt)
    http://blog.cleancoder.com/uncle-bob/2013/12/10/Thankyou-Kent.html

TDD

  • Kent Beck: Test Driven Development: By Example
    Kettő példán végigvezetve mutatja be Kent Beck a TDD lépéseit, hogyan dolgozik benne, milyen technikákat használ.
    https://www.amazon.com/Test-Driven-Development-Kent-Beck/dp/0321146530
  • CleanCoders videósorozat TDD epizódjai
    https://cleancoders.com/videos/clean-code/fundamentals
    https://cleancoders.com/videos/clean-code/advanced-tdd
  • (Advanced) Steve Freeman, Nat Pryce: Growing Object-Oriented Software, Guided by Tests
    A TDD londoni iskolajanak bemutatoja. Mock alapu megkozelites, egy komplett, nagyra novo rendszeren bemutatva a TDD gyakorlatait.
    https://www.amazon.com/Growing-Object-Oriented-Software-Guided-Tests/dp/0321503627
  • (Advanced) Gerard Meszaros: xUnit Test Patterns: Refactoring Test Code
    https://www.amazon.com/xUnit-Test-Patterns-Refactoring-Code/dp/0131495054/

Refactoring

  • Martin Fowler: Refactoring: Improving the Design of Existing Code
    A refaktorálás mint gyakorlat szerepe és fontossága a szoftverfejlesztésben. A könyv tartalmazza az azonosított refaktorálások katalógusát is.
    https://www.amazon.com/Refactoring-Improving-Design-Existing-Code/dp/0201485672/
