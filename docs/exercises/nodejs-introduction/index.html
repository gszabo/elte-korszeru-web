NodeJS bevezető gyakorlat

Az óra célja, hogy megismertessen a NodeJS alapjaival, a tárgy megértéséhez szükséges szinten. Az oktatóval közösen haladva érintünk néhány témakört, amelyek remélhetőleg elég alapot adnak a további önálló ismerkedéshez.

Ez a dokumentum az érintett témaköröket és fogalmakat sorolja csupán fel, sorvezető céllal. Az óra során kerülnek ezek részletesebb kifejtésre. Az önálló haladáshoz, vagy a gyakorlat megismétléséhez a megoldásokat érdemes böngészni. A kódfájlok kommentjeiben találhatóak az aktuális koncepció magyarázatai.

1. Telepítés ellenőrzése, fájl futtatása

  • Ellenőrizzük, hogy a NodeJS fel van telepítve a számítógépre. Telepíthető a hivatalos oldalról, vagy Unix rendszerek alatt az nvm segítségével.
  • Készítsünk egy egyszerű JavaScript fájlt, és futtassuk le NodeJS alatt.

2. NodeJS modulrendszer

  • Fájl betöltése
  • Mappa betöltése
  • node_modules használata
  • Module caching

3. Néhány beépített modul

  • Fájlok szinkron beolvasása (text és JSON)
  • Fájlok aszinkron beolvasása
  • path beépített modul
  • url beépített modul

4. Környezeti változók és parancssori argumentumok

  • Parancssori argumentumok átvétele
  • Környezeti változók használata

5. Az npm használata

  • init
  • install - csomag telepítése a hatalmas registryből
  • run script, start, test

6. Tesztelés

  • Egyszerű, egyenlőség alapú ellenőrzés
  • Néhány chai matcher használata ellenőrzésre
  • Lebegőpontos számok ellenőrzése toleranciával
  • Hibás inputok tesztelése
  • mocha watch és reporter paraméterek
